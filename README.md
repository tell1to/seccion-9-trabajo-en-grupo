## Trabajo Realizado

En este proyecto hemos explorado los fundamentos esenciales para el desarrollo de aplicaciones en Flutter. A lo largo del proceso aprendimos a manejar la **navegación entre pantallas**, utilizar **nuevos widgets**, trabajar con **botones y sus variantes**, así como crear **botones y tarjetas personalizadas**. También profundizamos en el uso de **Align**, **rutas propias de Flutter**, **Go_Router**, **paths**, y diversas configuraciones de enrutamiento, tanto nativas como de terceros. El objetivo principal fue sentar bases sólidas para seguir avanzando en siguientes secciones con una variedad de widgets y así comprender mejor cómo construir aplicaciones completas y bien estructuradas.
